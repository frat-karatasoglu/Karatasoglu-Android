## ‚ú® Key Features

Based on the latest updates (Practice 6), the application includes:

* **Browse Movies:** Displays a list of popular movies with thumbnails, titles, release years, and IMDb ratings.
* **Search Functionality:** Real-time search bar to find specific movies.
* **Detailed View:** Clicking a movie shows full description, cast info, and rating options.
* **Advanced Filtering:** Users can filter movies by:
    * **Genre:** (Action, Comedy, Drama, Sci-Fi, etc.)
    * **Rating Score:** Using an interactive slider.
* **Favorites System:** Ability to add movies to a "Favorites" list for quick access.
* **Bottom Navigation:** Easy switching between Movies, Favorites, Filters, and Profile screens.
